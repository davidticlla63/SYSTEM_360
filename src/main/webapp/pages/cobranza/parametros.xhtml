PARAMETROS DE COBRANZA

Las ventas disminuyen el stock de un Producto de Un Almacen.

Ayuda

La configuración de estos parametros es referencial.

- Seleccione la opción Generar Varias Orden de Salida para que el proceso de realizar la venta, genere diferentes Orden de Salida para cada almacen de donde pertenecen los productos.

- Seleccione la opción Generar Ordenes de Traspaso para que el proceso de realizar la venta, genere Ordenes de Traspaso de los productos a un solo almacen y generar una sola Orden de Salida.

Tipo de Salida de Almacén
* [ Tomar en cuenta que la opción selecionada sólo se aplicará cuando la venta tenga varios productos de diferentes almacenes. ]
